# UserCentricMileStoneProject
## Overview

A website based based on a British rock band called Reef. This site is consists of a Home page, a biography of the band - including a brief history of the band and an intro to it's members, a tour page, a media page - which will contain pictures and videos, a store page selling the bands merchandise and finally, a contact page where the site visitors can book the band.

The main purpose of the site is for existing fans but also for new fans are discovering Reef for the first time, who may wish to find out more about the band and perhaps book them for their own events.
